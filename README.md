# bunnynet-node-sdk

A NodeJS SDK for [Bunny.Net CDN](https://bunny.net/).
 
## Develop

Create a `.env` file in the root of the project:

```env
BUNNY_NET_PULLZONE=
BUNNY_NET_ACCESS_KEY=
```
